STANDARD TEST OBJECT:
{
  id: 'XXX-001',           // Unique identifier (e.g., WAT-001, CRP-021)
  name: 'Test Name',       // Full descriptive name
  price: 85,               // Price in USD
  days: 2,                 // Turnaround time in days
  critical: true/false,    // Critical test indicator (optional)
  unit: 'ppm',            // Measurement unit (optional)
  description: 'Text'      // Full description (optional)
}

CART ITEM (after addToCart):
{
  ...test,                 // All original test properties
  cartId: 'CART-{timestamp}-{random}',  // Unique cart identifier
  addedAt: ISO timestamp   // When added to cart
}

ORDER OBJECT:
{
  orderId: 'ORD-{timestamp}-{random}',
  orderNumber: 1001,       // Sequential
  receiptNumber: 'RCP-1001-2025',
  date: ISO timestamp,
  timestamp: Unix timestamp,
  customer: {
    name, email, company, phone,
    address, city, state, zip
  },
  payment: {
    method: 'card' | 'paypal',
    last4: '1234',
    status: 'completed',
    transactionId: 'TXN-{timestamp}'
  },
  items: [test objects],
  pricing: {
    subtotal: 850.00,
    tax: 68.00,
    total: 918.00
  },
  status: 'confirmed',
  trackingNumber: 'TRK-{timestamp}-{random}',
  estimatedCompletion: ISO timestamp (7 days out)

                    WORKFLOW: ADD TO CART → PURCHASE
═══════════════════════════════════════════════════════════════════════════════

1. USER SELECTS MODULE
   - Navigate from home or nav bar to Water/Soil/AG/etc.

2. USER BROWSES TESTS
   - Click featured category card
   - View expandable subcategories
   - See test details (ID, name, price, days, description)

3. USER ADDS TO CART
   - Click "+ Add to Cart" button
   - Success sound plays
   - Floating cart widget appears (if first item)
   - Widget shows live count & total

4. USER CLICKS CART WIDGET OR NAV
   - Navigate to /cart
   - See all items in detailed list
   - Can remove individual items
   - View subtotal, tax, total in sidebar

5. USER PROCEEDS TO CHECKOUT
   - Click "Proceed to Checkout" button
   - Navigate to /checkout

6. CHECKOUT STEP 1: REVIEW
   - See all cart items one more time
   - View order summary (subtotal, tax, total)
   - Click "Continue"

7. CHECKOUT STEP 2: CUSTOMER INFO
   - Fill out name, email (required)
   - Fill out phone (required)
   - Optional: company, address, city, state, zip
   - Form validation on submit
   - Click "Continue"

8. CHECKOUT STEP 3: PAYMENT
   - Select payment method (Card/PayPal)
   - For Card: Enter name, number (auto-formatted), expiry (MM/YY), CVV
   - Form validation
   - See final summary (subtotal, tax, total)
   - Click "Pay "

9. ORDER CREATED
   - createOrder() function generates:
     * Order Number (e.g., #1001)
     * Receipt Number (e.g., RCP-1001-2025)
     * Order ID (e.g., ORD-1698523456789-AB3CD)
     * Tracking Number (e.g., TRK-1698523456789-XY12Z)
   - Order saved to localStorage
   - Cart cleared

10. CONFIRMATION DISPLAYED
    - Large green checkmark
    - "Order Confirmed!" message
    - Display Order #, Receipt #, Order ID, Tracking #
    - Display total paid
    - Display date/time
    - Buttons: "Back to Home" | "View My Orders"

11. VIEW ORDERS PAGE
    - Navigate to /orders
    - See analytics: Total Orders, Revenue, Average
    - See all orders (newest first)
    - Each order shows: #, items, customer, payment, total
    - Can click items to see details
